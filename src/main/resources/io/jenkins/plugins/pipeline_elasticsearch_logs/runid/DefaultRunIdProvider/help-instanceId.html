A unique string to identify your Jenkins instance.
Leave empty to use the internal Jenkins instance ID (recommended).
When you change this after you executed first builds, the logs of builds executed before the change will no longer by found.

The instance ID is stored in Elasticsearch entries as runId.instance. Here you find a full example:
"runId": {
  "instance": "jenkins1",
  "project":  "TestPipeline",
  "build":    7,
  "uid:       "OGQyY2Q1YjMtOGE2Zi00NDYzLTkwYm_5"
}
